Breadboard Power Supply

A dual-output 5V/3.3V power supply for breadboard prototyping with LEDs.

Overview

This breadboard power supply provides regulated 5V and 3.3V outputs from a 6–12V DC input. It is designed for easy connection to standard breadboards, providing reliable voltage for small electronic projects.

Project Details

The 5V output uses an LM7805 linear regulator with input and output capacitors for stability, while the 3.3V output is generated using an adjustable LM317 regulator with resistors setting the voltage to 3.3V. A power LED indicates when input power is present.

The outputs are selectable via jumpers or connectors for use with breadboards, jumper wires, or screw terminals, allowing flexible prototyping setups.

What I Learned

  • Voltage Regulation: Implementing stable 5V and 3.3V outputs using LM7805 and LM317 regulators.
  • Power Indication: Adding LEDs to show input power and output status for improved usability.
  • Output Flexibility: Designing selectable outputs for breadboards, jumper wires, or terminals.

Summary of Features

  • Input: 6–12V DC
  • Outputs: 5V regulated, 3.3V regulated
  • Indicators: Power LED for input
  • Selectable Outputs: Breadboard, jumper wires, screw terminals
  • Compact design for breadboard compatibility
