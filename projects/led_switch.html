LED Switch PCB

A beginner PCB project to design a simple on/off LED switch using KiCad.

Overview

This was my first PCB, created as a way to learn schematic capture and PCB layout. The board consists of a switch, an LED, and a current-limiting resistor. I learned how to create and link footprints, run design rule checks, and prepare fabrication files. Tools: KiCad Skills Gained: Schematic design, PCB layout basics, DRC checking
